# 🛒 Projeto Site de Produtos (HTML)

Este é um projeto simples desenvolvido utilizando apenas **HTML**, com o objetivo de praticar a estruturação de páginas web e navegação entre telas.

---

## 📌 Funcionalidades

- 🏠 Página inicial
- 📦 Página de produtos
- 📞 Página de contato
- 🔗 Navegação entre páginas
- 🖼️ Exibição de imagens de produtos

---

## 🛠️ Tecnologias utilizadas

- HTML5

---

## 🎯 Objetivo do projeto

Praticar:

- Estrutura básica de páginas HTML
- Uso de links entre páginas
- Organização de conteúdo
- Inserção de imagens
- Criação de um layout simples

---

## 📂 Estrutura do projeto

📁 projeto
│-- index.html
│-- produtos.html
│-- contato.html
│-- imagens/


---

## 🚀 Como executar o projeto

1. Baixe o repositório
2. Abra o arquivo `index.html` no navegador
